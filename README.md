# Neural-Network-Project
Multi-Layer Perceptron Neural Network

**Overview:** 
This project implements a Multi-Layer Perceptron (MLP) neural network using Python and NumPy. The model performs binary classification and includes custom implementations of forward propagation, backpropagation, gradient descent, loss calculation, and performance evaluation.
**Features:** Built neural network architecture without machine learning frameworks, implemented forward and backward propagation, binary cross-entropy loss function, gradient descent optimization, feature standardization. F1-score performance evaluation, and hyperparameter experimentation with hidden layer sizes and learning rates.
**Technologies:**
Python, 
NumPy, and
Matplotlib.
**Experiments:**
The project evaluates model performance by varying:
Hidden layer sizes (5, 25, 50, and 100 units), 
Learning rates (1.0, 0.1, 0.01, and 0.001), 
Performance was analyzed using training loss and F1-score metrics, with results visualized through comparison graphs.
**Files:**
Main.py – Neural network implementation and training pipeline, 
plot.py – Visualization of training results, 
magic_train.txt – Training dataset, 
magic_test.txt – Testing dataset, 
*.npy – Stored experiment results, and
*.png – Performance graphs and visualizations.
**Key Learning Outcomes:** Understanding neural network fundamentals, implementing backpropagation from scratch, hyperparameter tuning and model evaluation, and working with numerical computing and data preprocessing.
